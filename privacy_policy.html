1. はじめに

Chrono Ride（以下「本アプリ」）は、NARUKAMI AKUT が提供するサイクリング向けダッシュボードアプリです。本ポリシーは、本アプリがどのような情報を扱うかを説明します。

2. 収集する情報

情報の種類用途外部送信
位置情報（GPS） スピード・距離・高度のリアルタイム表示 高度取得時のみ（国土地理院）
加速度センサーデータ 傾斜角のリアルタイム表示 なし（端末内のみ）
アプリ設定 速度単位・テーマなどの設定保存 なし（端末内のみ）
クラッシュ情報 不具合の検出・修正（Firebase Crashlytics） あり（Google）
匿名利用統計 機能改善のための利用状況把握（Firebase Analytics） あり（Google）
広告識別子 広告配信（Google AdMob） あり（Google）

本アプリはアカウント登録・ログイン機能を持たず、氏名・メールアドレス等の個人識別情報を収集しません。高度タイル使用時、精度向上のために現在地の緯度・経度を国土地理院の標高APIに送信します。それ以外のセンサーデータは端末内でのみ処理され、外部サーバーへの送信は行われません。

3. 第三者サービス

本アプリは以下の Google サービスを利用しています。各サービスの個人情報取り扱いについては、Google のプライバシーポリシーをご確認ください。

  • Firebase Crashlytics — クラッシュレポート収集
  • Firebase Analytics — 匿名の利用統計
  • Google AdMob — 広告配信（非プレミアムユーザー）
  • 国土地理院 標高API — 高度タイル使用時、精度向上のために緯度・経度を送信します（個人識別情報は含まれません）

4. 位置情報の取り扱い

本アプリは、スピード・距離・高度を表示するためにデバイスの位置情報（GPS）を使用します。高度タイルを使用する場合、精度向上のために現在地の緯度・経度を国土地理院の標高APIに送信することがあります。送信されるのは座標情報のみであり、個人を特定できる情報は含まれません。ネットワーク接続がない場合は端末のGPSから高度を取得します。アプリを閉じると位置情報の取得は停止します。

本アプリの使用中に生じた事故・怪我・財物の損害・その他いかなる損害についても、開発者は一切の責任を負いません。自転車・電動バイク等の乗車中は、安全に十分注意したうえで本アプリをご利用ください。走行中のスマートフォン操作は法律で規制されている場合があります。

本アプリは現状有姿（as-is）で提供されます。動作の継続性・正確性・特定目的への適合性について、明示・黙示を問わず一切保証しません。

Privacy Policy & Disclaimer

Last updated: May 18, 2026

1. Overview

Chrono Ride is a cycling dashboard app developed by NARUKAMI AKUT. This policy explains how the app handles information.

2. Information We Collect

Data TypePurposeTransmitted
Location (GPS) Real-time speed, distance, and altitude display Altitude only (GSI Japan)
Accelerometer data Real-time slope display No (on-device only)
App settings Storing preferences (units, theme, etc.) No (on-device only)
Crash reports Bug detection and fixes (Firebase Crashlytics) Yes (Google)
Anonymous usage stats Feature improvement (Firebase Analytics) Yes (Google)
Advertising identifier Ad delivery (Google AdMob) Yes (Google)

The app does not require an account or login and does not collect personally identifiable information such as name or email address. When the altitude tile is in use, the device's latitude and longitude are sent to the Geospatial Information Authority of Japan (GSI) Elevation API to improve accuracy. No personally identifiable information is included. All other sensor data is processed entirely on-device.

3. Third-Party Services

The app uses the following Google services. Please refer to Google's Privacy Policy for details on how they handle data.

  • Firebase Crashlytics — crash report collection
  • Firebase Analytics — anonymous usage statistics
  • Google AdMob — ad delivery (non-premium users)
  • GSI (Geospatial Information Authority of Japan) Elevation API — latitude/longitude sent to improve altitude accuracy (no personal data included)

4. Location Data

The app accesses device GPS to display speed, distance, and altitude. When the altitude tile is active, the current latitude and longitude are sent to the GSI Elevation API to improve accuracy. Only coordinate data is transmitted; no personally identifiable information is included. If no network connection is available, altitude is derived from the device's GPS sensor. Location access stops when the app is closed.

5. Disclaimer

Speed, distance, slope, and altitude values displayed by this app are estimates derived from GPS and device sensors. Accuracy is not guaranteed.
